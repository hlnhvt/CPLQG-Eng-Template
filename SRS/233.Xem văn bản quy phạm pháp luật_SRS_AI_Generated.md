## A.IV.1_Hệ thống văn bản pháp luật tiếng Anh
### CPLQG-EN-233 - Xem văn bản quy phạm pháp luật

#### 1. Mục đích

Cho phép NSD là người dùng nước ngoài, nhà đầu tư và doanh nghiệp quốc tế xem toàn bộ nội dung văn bản pháp luật Việt Nam bằng tiếng Anh. NSD bấm vào tên văn bản từ trang tìm kiếm (UC230) để mở trang chi tiết với 5 tab: Summary (tóm tắt metadata), Content (toàn văn PDF tiếng Anh), Validity (thông tin hiệu lực), Vietnamese (toàn văn PDF tiếng Việt) và Download (tải file).

##### a. Phân quyền

Hệ thống thực hiện phân quyền chức năng Xem văn bản quy phạm pháp luật theo nguyên tắc sau:

- Khách (chưa đăng nhập): Được xem toàn bộ thông tin và nội dung văn bản.
- Người dùng đã đăng nhập: Xem đầy đủ như khách.

##### b. Điều kiện thực hiện

- NSD bấm vào tên văn bản từ danh sách kết quả tìm kiếm (UC230.MH01) hoặc từ trang tóm tắt chính sách (UC232).
- Văn bản có bản dịch tiếng Anh hoặc file PDF tiếng Anh trong hệ thống.

---

#### CPLQG-EN-233.MH01 – Document Detail – Tab Summary

##### 1. Màn hình
Trang chi tiết văn bản pháp luật, hiển thị mặc định tại tab Summary. Header trang nền xanh đậm hiển thị tên văn bản và số hiệu chính thức. Bên dưới header là thanh tab gồm 5 tab: Summary (đang chọn), Content, Validity, Vietnamese, Download. Nội dung tab Summary gồm bảng metadata hai cột và đoạn tóm tắt nội dung văn bản. Giao diện hoàn toàn bằng tiếng Anh.

##### 2. Mô tả thông tin trên màn hình

| Trường thông tin | Kiểu dữ liệu | Bắt buộc | Mặc định | Mô tả |
| --- | --- | --- | --- | --- |
| **Header trang** | | | | |
| Document title | text | Không | - | Tên văn bản tiếng Anh, in đậm, màu trắng, nền xanh đậm |
| Official number | text | Không | - | Số hiệu chính thức của văn bản (ví dụ: "18/2013/L-CTN"), hiển thị màu xanh dương dưới tiêu đề |
| **Thanh tab** | | | | |
| Tab bar | tab | Không | Tab "Summary" active | Thanh tab gồm 5 mục: Summary / Content / Validity / Vietnamese / Download |
| **Nội dung tab Summary – Bảng metadata** | | | | |
| Issuing body | text | Không | - | Tên cơ quan ban hành văn bản |
| Official number | text | Không | - | Số hiệu chính thức của văn bản |
| Type | text | Không | - | Loại văn bản (ví dụ: Law, Decree, Circular, Resolution) |
| Issuing date | text | Không | - | Ngày ban hành |
| Effective date | text | Không | - | Ngày có hiệu lực |
| Fields | text | Không | - | Lĩnh vực pháp lý của văn bản (ví dụ: Land, Investment, Finance) |
| Gazette number | text | Không | - | Số Công báo đăng văn bản |
| Gazette date | text | Không | - | Ngày đăng Công báo |
| Signer | text | Không | - | Họ tên và chức danh người ký ban hành văn bản |
| Expiry date | text | Không | - | Ngày hết hiệu lực (nếu có) |
| Effect status | badge/text | Không | - | Trạng thái hiệu lực hiện tại (ví dụ: "In force" màu xanh lá, "Expired" màu xám, "Partially in force") |
| **Nội dung tab Summary – Tóm tắt** | | | | |
| Document summary | rich text | Không | - | Đoạn tóm tắt nội dung chính của văn bản bằng tiếng Anh |

##### 3. Chức năng trên màn hình

| STT | Tên chức năng | Định dạng | Mô tả |
| --- | --- | --- | --- |
| 1 | Xem thông tin tóm tắt văn bản | table + text (auto) | Hệ thống tự động hiển thị bảng metadata và đoạn tóm tắt khi NSD mở trang chi tiết. Không cần thao tác thêm |
| 2 | Chuyển tab | tab click | NSD bấm vào tên tab (Content / Validity / Vietnamese / Download) để chuyển sang nội dung tương ứng |

#### CPLQG-EN-233.MH02 – Document Detail – Tab Content

##### 1. Màn hình
Tab Content hiển thị toàn văn văn bản pháp luật bằng tiếng Anh dưới dạng PDF viewer nhúng. Bố cục gồm thanh công cụ PDF (toolbar) phía trên, panel điều hướng bên trái (thumbnail/bookmark/attachment) và khung xem PDF chính. Giao diện hoàn toàn bằng tiếng Anh.

##### 2. Mô tả thông tin trên màn hình

| Trường thông tin | Kiểu dữ liệu | Bắt buộc | Mặc định | Mô tả |
| --- | --- | --- | --- | --- |
| **Thanh công cụ PDF** | | | | |
| Page navigation | input + text | Không | "1 / [tổng số trang]" | Ô nhập số trang hiện tại và tổng số trang (ví dụ: "1 / 36"). Có nút Previous page và Next page |
| Zoom control | input + text | Không | "90%" | Mức zoom hiện tại. Có nút Zoom in và Zoom out, hoặc nhập trực tiếp giá trị phần trăm |
| Rotate | icon button | Không | - | Nút xoay trang PDF |
| Download | icon button | Không | - | Nút tải file PDF tiếng Anh |
| Print | icon button | Không | - | Nút in tài liệu PDF |
| More options | icon button | Không | - | Nút "..." mở thêm tùy chọn PDF |
| **Panel điều hướng (trái)** | | | | |
| Thumbnail panel | icon button | Không | - | Icon xem thumbnail trang trong panel trái |
| Bookmark panel | icon button | Không | - | Icon xem mục lục/bookmark của tài liệu |
| Attachment panel | icon button | Không | - | Icon xem tệp đính kèm trong tài liệu |
| **Khung xem PDF** | | | | |
| PDF viewer | embedded PDF | Không | - | Khung hiển thị nội dung PDF toàn văn văn bản tiếng Anh |

##### 3. Chức năng trên màn hình

| STT | Tên chức năng | Định dạng | Mô tả |
| --- | --- | --- | --- |
| 1 | Xem toàn văn PDF tiếng Anh | PDF viewer (auto) | Hệ thống nhúng file PDF tiếng Anh và hiển thị ngay khi NSD chọn tab Content |
| 2 | Điều hướng trang | input/button | NSD bấm Previous/Next page hoặc nhập số trang để điều hướng đến trang mong muốn |
| 3 | Phóng to / Thu nhỏ | icon button | NSD bấm Zoom in/Zoom out hoặc nhập giá trị zoom để thay đổi kích thước hiển thị trang PDF |
| 4 | Tải PDF tiếng Anh | icon button | NSD bấm nút Download trên toolbar. Hệ thống tải file PDF tiếng Anh về máy |
| 5 | In tài liệu | icon button | NSD bấm nút Print. Hệ thống mở hộp thoại in tài liệu PDF |
| 6 | Xem thumbnail trang | panel | NSD bấm icon Thumbnail trong panel trái để xem danh sách thu nhỏ các trang PDF |
| 7 | Xem bookmark/mục lục | panel | NSD bấm icon Bookmark để xem mục lục tài liệu và bấm mục để nhảy đến phần tương ứng |

#### CPLQG-EN-233.MH03 – Document Detail – Tab Validity

##### 1. Màn hình
Tab Validity hiển thị thông tin hiệu lực pháp lý của văn bản. Bao gồm tiêu đề "Validity", mô tả trạng thái hiệu lực bằng chữ nghiêng, thẻ văn bản (document card) kèm nút tải và khu vực thông tin ngày tháng hiệu lực. Giao diện hoàn toàn bằng tiếng Anh.

##### 2. Mô tả thông tin trên màn hình

| Trường thông tin | Kiểu dữ liệu | Bắt buộc | Mặc định | Mô tả |
| --- | --- | --- | --- | --- |
| **Tiêu đề tab** | | | | |
| "Validity" heading | text | Không | "Validity" | Tiêu đề phần, in đậm |
| Validity description | text (italic) | Không | - | Mô tả ngắn về trạng thái hiệu lực của văn bản, in nghiêng |
| **Thẻ văn bản** | | | | |
| Document card – Title | text | Không | - | Tên văn bản, hiển thị trong thẻ |
| Document card – Download button | button | Không | - | Nút "↓ Download" màu xanh dương, bấm để tải file PDF văn bản |
| **Thông tin hiệu lực** | | | | |
| Issuing date | text | Không | - | Nhãn "Issuing date:" + ngày ban hành |
| Effective date | text | Không | - | Nhãn "Effective date:" + ngày có hiệu lực |
| Effect status | badge | Không | - | Trạng thái hiệu lực (ví dụ: "In force" màu xanh lá) |
| Update date | text | Không | - | Nhãn "Update date:" + ngày cập nhật thông tin gần nhất |

##### 3. Chức năng trên màn hình

| STT | Tên chức năng | Định dạng | Mô tả |
| --- | --- | --- | --- |
| 1 | Xem thông tin hiệu lực | text (auto) | Hệ thống hiển thị trạng thái và các ngày hiệu lực khi NSD chọn tab Validity |
| 2 | Tải văn bản | button | NSD bấm "↓ Download" trong document card. Hệ thống tải file PDF văn bản về máy |

#### CPLQG-EN-233.MH05 – Document Detail – Tab Download

##### 1. Màn hình
Tab Download hiển thị danh sách các file tài liệu có thể tải về, phân theo ngôn ngữ. Mỗi file hiển thị tên file và nút Download. Giao diện hoàn toàn bằng tiếng Anh.

##### 2. Mô tả thông tin trên màn hình

| Trường thông tin | Kiểu dữ liệu | Bắt buộc | Mặc định | Mô tả |
| --- | --- | --- | --- | --- |
| **Danh sách file tải về** | | | | |
| Download file list | list | Không | - | Danh sách các file tài liệu có thể tải. Mỗi dòng gồm: tên file (bao gồm ngôn ngữ và định dạng) và nút Download |
| File item – Filename | text | Không | - | Tên file tài liệu, hiển thị ngôn ngữ và loại file (ví dụ: tên văn bản + "[English]" hoặc "[Vietnamese]" + ".pdf") |
| File item – Download button | button | Không | - | Nút "↓ Download" màu xanh dương. Bấm để tải file về máy |

##### 3. Chức năng trên màn hình

| STT | Tên chức năng | Định dạng | Mô tả |
| --- | --- | --- | --- |
| 1 | Xem danh sách file tải về | list (auto) | Hệ thống hiển thị danh sách tất cả file có thể tải khi NSD chọn tab Download. Phân biệt theo ngôn ngữ (tiếng Anh, tiếng Việt) |
| 2 | Tải file PDF tiếng Anh | button | NSD bấm "↓ Download" tương ứng với file bản tiếng Anh. Hệ thống tải file PDF tiếng Anh về máy NSD |
| 3 | Tải file PDF tiếng Việt | button | NSD bấm "↓ Download" tương ứng với file bản tiếng Việt. Hệ thống tải file PDF tiếng Việt về máy NSD |
